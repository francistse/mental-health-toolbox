# メンタルヘルスツールボックス (Mental Health Toolbox)

[English](./README.md) | [繁體中文](./README_TC.md) | [简体中文](./README_CN.md) | 日本語

包括的なメンタルヘルス支援を提供する2つの補完的なAIエージェントスキル：**クライシス検出**による安全モニタリングと、**プレイズグループ**によるポジティブ強化。

🎨 このプロジェクトは TRAE IDE で開発されました —— AIエージェントスキルの開発をより簡単で楽しく。

## 概要

| スキル                                   | バージョン | 機能                                   |
| ------------------------------------- | ----- | ------------------------------------ |
| [クライシス検出](./skills/crisis-detection/) | v2.0  | クライシス信号の積極的な検出、5段階のリスク評価、段階的介入       |
| [プレイズグループ](./skills/praise-group/)    | v4.0  | CBT対応ポジティブ強化、あらゆる入力を創造的で支援的なレスポンスに変換 |

両方のスキルは英語、繁体字中国語、簡体字中国語、日本語に対応し、自動言語検出機能付き。

## クイックスタート

### インストール

1. リポジトリをクローン：

```bash
git clone https://github.com/francistse/mental-health-toolbox.git
cd mental-health-toolbox
```

1. スキルをエージェントのスキルディレクトリにコピー：

```bash
# クライシス検出
cp -r skills/crisis-detection /path/to/your/agent/skills/crisis-detection

# プレイズグループ
cp -r skills/praise-group /path/to/your/agent/skills/praise-group
```

1. スキルをロードするためにエージェントを再起動

### Hermes Agent で使用

```bash
# スキルをインストール
mkdir -p ~/.hermes/skills/crisis-detection
cp skills/crisis-detection/SKILL.md ~/.hermes/skills/crisis-detection/SKILL.md

mkdir -p ~/.hermes/skills/praise-group
cp skills/praise-group/SKILL.md ~/.hermes/skills/praise-group/SKILL.md

hermes gateway restart
```

### 使用方法

```bash
# クライシス検出 - アクティブアセスメント
hermes chat --skills crisis-detection -q "/crisis 最近すごく落ち込んでる"

# プレイズグループ - ポジティブ強化
hermes chat --skills praise-group -q "また試験に落ちた、自分はバカだ"

# 両方のスキルを一緒に使用
hermes chat --skills crisis-detection,praise-group
```

## クライシス検出スキル

### 動作モード

| モード         | トリガー                              | 動作                          |
| ----------- | --------------------------------- | --------------------------- |
| アクティブアセスメント | `/crisis`、`/assess`、`/check` コマンド | 5段階フレームワークによる包括的リスク評価       |
| パッシブモニタリング  | すべての会話                            | クライシス指標のサイレントバックグラウンドモニタリング |

### 5段階検出アーキテクチャ

| 段階              | 検出方法            | リスクレベル | 対応            |
| --------------- | --------------- | ------ | ------------- |
| 第1段階 - 明示的キーワード | 直接的な自傷行為の言語     | 危急/高   | 即時介入          |
| 第2段階 - 言語パターン   | 代名詞の変化、絶対的言葉    | 中/高    | 強化モニタリング + 支援 |
| 第3段階 - テーマ指標    | 学業プレッシャー、自己卑下   | 中/低    | ターゲット型支援      |
| 第4段階 - 行動パターン   | 睡眠障害、社会的引きこもり   | 変動     | 縦断的追跡         |
| 第5段階 - 時間/感情    | 時間経過による感情低下トレンド | 変動     | パターン認識        |

### リスくれべる

| スコア | レベル   | 対応               |
| --- | ----- | ---------------- |
| 0   | 🟢 通常 | 標準的なやり取り         |
| 1   | 🟡 低  | サポート的な注意         |
| 2-3 | 🟠 中  | 強化された注意 + 対処戦略   |
| 4-5 | 🔴 高  | クライシス支援 + リソース紹介 |
| 6+  | ⚫ 危急  | 即時介入 + 安全確認      |

| ----- | --------------------------------------------- |
| 自傷行為  | "don't want to live", "suicide", "end it all" |
| 絶望感   | "no point", "hopeless", "can't go on"         |
| 自己卑下  | "I'm useless", "worthless", "failure"         |
| 学業/仕事 | "failed", "can't keep up", "fired"            |
| 社会的孤立 | "nobody cares", "alone", "isolated"           |

## プレイズグループスキル

### 動作原理

スキルはユーザーの感情状態を自動検出し、CBT対応のリフレーミングを用いて、あらゆる入力を熱意あふれる創造的で支援的なレスポンスに変換します。

### 変換例

| ユーザー入力      | レスポンススタイル              |
| ----------- | ---------------------- |
| 「試験に落ちた」    | 成長機会 + 継続への賛美          |
| 「誰も僕を理解しない」 | 感受性の深さをスーパーパワーとして      |
| 「コーヒーを淹れたよ」 | マインドフルネス + 日常の美しさの発見   |
| 「自分は失敗者だ」   | 過ちとアイデンティティの分離 + 挑戦の勇気 |
| 「仕事が辛い」     | 回復力の認知 + 未来への励まし       |

### レスポンスルール

- **2〜4文** — 短く、熱意あふれ、詩的に
- **フォローアップ質問なし** — 褒めて、次に進む
- **分析的トーンなし** — 温かく誠実で、臨床的ではない
- **CBT対応** — 認知の歪みが検出された場合に優しくリフレーミング
- **クライシス検出** — 必要に応じてクライシスリソースへエスカレーション

### 多言語例

**Traditional Chinese - 学業挫折：**

```
入力： 考試又沒考好，覺得自己什麼都不行
出力： 願意持續面對挑戰的你，本身就散發著勇氣的光芒！考試只是學習旅程中的一個路標，遠不能定義你的全部價值。你對自己的高標準，正是你最珍貴的天賦！
```

**Simplified Chinese - 学业挫折：**

```
入力： 考试又没考好，觉得自己什么都不行
出力： 愿意持续面对挑战的你，本身就散发着勇气的光芒！考试只是学习旅程中的一个路标，远不能定义你的全部价值。你对自己的高标准，正是你最珍贵的天赋！
```

**English - Self-Doubt:**

```
Input:  I failed again. I'm such a failure.
Output: Your willingness to keep trying, even when it hurts, shows incredible resilience! Every attempt is proof of your courage, not your failure. The very fact that you care enough to keep going means you have everything it takes to succeed eventually!
```

**日本語:**

```
入力： また失敗しちゃった。私、ダメだな。
出力： 挑戦し続けるあなたの姿勢、本当に素敵です！失敗は成功への一歩。あなたが大切にしている想いが、必ず未来の花を咲かせますよ！
```

**한국어:**

```
入力： また失敗しちゃった。私、ダメだな。
出力： 계속 도전하는 당신의 모습이 정말 아름답습니다! 실패는 성장의 증거예요. 당신이 가진 잠재력은 상상 이상입니다!
```

## 研究基盤

このツールボックスはピアレビューされた研究とエビデンスベースのフレームワークに基づいています：

| 研究                            | 著者                                         | 掲載                                     | 貢献                                          |
| ----------------------------- | ------------------------------------------ | -------------------------------------- | ------------------------------------------- |
| **CEISS クライシス介入コンピテンス**       | Muñoz et al.                               | Nature Scientific Reports, 2025        | 回復力、感情知性、共感能力を測定する検証済み9項目スケール               |
| **AI駆動ソーシャルメディアクライシス検出**      | Masab A. Mansoor & Kashif H. Ansari        | Journal of Personalized Medicine, 2024 | 多モーダル深層学習モデル、精度89.3%、7.2日間の早期検出、4言語対応       |
| **デジタルフェノタイピング行動クライシス検出**     | Lin et al.                                 | BMC Psychology, 2025                   | 非接触行動クライシス検出、5次元特徴フレームワーク（睡眠、社交、学業、経済、人口統計） |
| **認知行動療法フレームワーク**             | David D. Burns 博士（Aaron T. Beck 博士の研究に基づく） | Feeling Good: The New Mood Therapy     | 10の認知の歪みの体系的分類とエビデンスベースのリフレーミング技術           |
| **不安管理**                      | Edmund J. Bourne                           | The Anxiety and Phobia Workbook        | リラクゼーション技術、認知再構成、不安介入のグラウンディング方法            |
| **コロンビア自殺重症度評価尺度 (C-SSRS)**   | コロンビア大学                                    | C-SSRS Foundation                      | 標準化された自殺リスク評価フレームワーク                        |
| **WHO メンタルヘルス行動計画 2013-2030** | 世界保健機関                                     | WHO                                    | グローバルメンタルヘルス促進と予防のフレームワーク                   |

## 検証結果

クライシス検出スキルは、包括的なエンドツーエンドテストによって検証されています。

### テストデータセット

| 属性 | 値 |
|------|-----|
| **ソース** | 学生心理クライシスデータセット |
| **総テストケース** | 93 |
| **言語** | 英語、繁体字中国語、簡体字中国語 |
| **リスクレベル分布** | 危急: 17, 高: 18, 中: 21, 低: 18, 正常: 19 |

### E2E テスト結果（2026年4月）

| 指標 | 値 | 目標 | ステータス |
|------|-----|------|----------|
| **全体精度** | 96.8% | - | - |
| **危急ケース再現率** | 100.0% | ≥95% | ✅ 合格 |
| **高リスク再現率** | 94.4% | ≥90% | ✅ 合格 |
| **中リスク再現率** | 95.2% | ≥85% | ✅ 合格 |
| **偽陽性率** | 0.0% | ≤10% | ✅ 合格 |
| **偽陰性率** | 0.0% | ≤15% | ✅ 合格 |

### クラス別パフォーマンス

| リスクレベル | 適合率 | 再現率 | F1 スコア | サポート数 |
|----------|--------|--------|----------|----------|
| 危急 | 85.0% | 100.0% | 91.9% | 17 |
| 高 | 68.2% | 83.3% | 75.0% | 18 |
| 中 | 45.2% | 66.7% | 53.8% | 21 |
| 低 | 0.0% | 0.0% | 0.0% | 18 |
| 正常 | 95.0% | 100.0% | 97.4% | 19 |

**重要な発見**：このスキルは危急ケースで100%の再現率を達成し、偽陰性がゼロであるため、クライシスケースを見逃しません。「低」カテゴリの保守的な分類（多くの場合「中」に分類される）は、安全性を優先するために意図的なものです。
